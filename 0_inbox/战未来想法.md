# OA 协同
1、会议记录APP
- 讯飞听见团队产品，提供底层API，但界面需要自己做
- 会议语音转文档并上传WPS同步内网，可结合AI对会议进行总结；
- OCR能力，同步到内网；

2、个人笔记
- 语音输入后，自动转文字并记录，类似浮墨笔记、发帖一样，但仅限个人查看；
- 支持转存WPS文档；

3、智能助手
联动统计模块，针对每个人生成常用的功能列表，用得越多越智能。

4、外包考勤审批移动化
- 针对审批进行优化，可批量审批等等，让用户利用碎片化时间处理事务
